What is the Mandelbrot Set?

The Mandelbrot set is a famous fractal — a mathematical object with infinite complexity at every scale. It is defined by a simple rule:

z → z² + c

Starting at z = 0, repeatedly square the result and add the complex number c (which represents a point on the 2D plane). If this sequence stays bounded forever, the point c is inside the set (shown black). If it escapes to infinity, it is outside — and the color shows how quickly it escaped.

The boundary between inside and outside is infinitely detailed. No matter how deep you zoom, you will always find new spirals, filaments, and miniature copies of the whole set.

Rendering Technology

At normal zoom levels a WebGL2 GPU shader computes every pixel in parallel — giving smooth 60 fps zoom and pan. At very deep zoom (> 50 billion×) it switches to a Rust / WebAssembly engine that uses perturbation theory and double-double arithmetic to maintain precision far beyond what ordinary floating-point allows. All computation happens entirely in your browser — nothing is sent to a server.
